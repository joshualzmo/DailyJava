# Day 21 - Library System

## Topic
In today's program I practiced creating a custom exception class.

## Description
This program allows users to simulate checking out books from the library of 5 books. Users
search for the books by name, if not found, then the custom exception is thrown.

## Concepts Used
- Classes & objects
- Exceptions
- Try/catch
- Loops
- Arrays

## What I Learned
I learned how to create a custom exception by creating a child class from the already exsisting exeption class
