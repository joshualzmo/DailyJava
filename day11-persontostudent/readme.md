# Day 11 — Person to Student

## Topic
In today's project I practiced the nasics of inheritance by extending a class and by using the 'super' keyword. I also
practiced throwing exceptions and "error-proofing" my code.

## Description
This program allows users to create student objects which are extensions of the Person class. Users can get basic student 
information from the methods created.

## Concepts Used
- Classes & objects
- Ingeritance
- Super
- Throwing Exceptions

## What I Learned
I learned how to extend a class using the super keyword.
