# Day XX — Student Grade Calculator

## Topic
In today's program I practiced throwing exceptions and using try/catch block in the main to handle errors
## Description
This program prompts users to create a student object. Users are asked for the name, student id, and the grades for 
each student which are stored in an array.
## Concepts Used
- Classes & objects
- Wrapper (Double)
- Try/Catch
- Loops

## What I Learned
I used the wrapper class of double to initialize the array. I also used a try/catch block within a loop to check each 
index.

## How to Run
1. Compile the file(s)
2. Run the main class
