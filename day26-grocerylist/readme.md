# Day 26 — Grocery List

## Topic
In today's program, I practiced creating algorithms to add nodes to the front and end of a linked list.
## Description
This program allows user to add grocery items to the start or end of a list.
## Concepts Used
- Classes & objects
- Linked lists
- Loops
- Getters and Setters

## What I Learned
I learned that you must traverse the entire list to access the last node causing O(n) time complexity
for certain algorithms
