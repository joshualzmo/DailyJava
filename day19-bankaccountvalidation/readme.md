# Day XX — Bank Account Validation

## Topic
In today's program I practiced verifying input by throwing exception in a class. I also practiced taking user
input using scanner

## Description
This program simulates a bank where a user can withdrawl or deposit money. Accounts are stored in an array 
and can be accessed through the bank account number

## Concepts Used
- Classes & objects
- Scanner 
- Throwing exceptions
- Loops 

## What I Learned
I learned how to throw exceptions in a class and to collect user input using scanner
